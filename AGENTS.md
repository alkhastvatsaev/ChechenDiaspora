# 🌍 Project Mission: Global Chechen Diaspora Hub

## 🎯 Core Directive
The mission of this Progressive Web App (PWA) is to create a high-leverage, community-driven "Contact Book" for the entire Chechen diaspora. It is designed to be an open-source hub where members can find and connect with experts within the community (lawyers, notaries, politicians, entrepreneurs, etc.) to help the diaspora flourish globally.

## ⚙️ Core Logic (ALWAYS FOLLOW)
- **High Leverage & Community Value**: Every feature should prioritize helping Chechen people find and help each other.
- **Expertise Visibility**: Focus on highlighting skills and professions (Politicians, Lawyers, Notaries, etc.) to enable quick problem-solving 
- **Shared Access Control**: Access is protected by a community passphrase to maintain a balance of openness for the community and security from external entities.
- **Database Growth**: The logic must facilitate constant expansion. New members can join and also propose existing contacts to be added.
- **No Fillers**: Do not add "social" fluff. This is a functional tools for networking and community strength.

## 🤝 Project Philosophy
- **Leverage**: Using the collective power of 2000+ existing contacts to benefit every person in the diaspora.
- **Persistence**: Never lose sight of the "Bridge" function of this app (Chechen-to-Chechen connection).
- **Expansion**: A recursive cycle where helping people attracts more people, which in turn brings more contacts.
